    1. private $username = 'nonono'; 和 private $password = 'yesyes';：类中的私有成员变量 $username 和 $password 被初始化为默认值。
    1. public function __construct($username, $password)：该构造函数接受两个参数 $username 和 $password，并将它们赋值给对应的成员变量。通过构造函数，实例化对象时可以传入不同的用户名和密码。

一句话木马
2023/10/16

Level_1

最简单的一个php一句话木马：

  • @不进行报错
  • php中有几个超全局变量，$_POST是其中一个。
    $_POST['shell']意为用POST的方法接收shell变量。
  • eval()函数将字符串作为PHP代码执行
    连起来，就是，用POST的方法接收shell变量，然后用eval()函数将变量shell作为php代码执行，并且该过程中即使出现执行错误也不报错。shell作为一个变量，可以`shell=xxx`，当我们上传该木马成功后，我们再通过POST上传对变量shell的更改，比如`shell=system("uname -a)`，会得到该命令执行后的响应。

蚁剑等工具连接原理（一句话密码的真相）

在蚁剑之类的工具进行连接的时候，我们只是输入shell当作密码，实则是蚁剑自动对shell变量进行了木马代码赋值并执行，让我们连接上了服务器。（具体实践可用burpsuite对蚁剑进行抓包）

下载Module

有时候使用py的时候会报错说xx module没有，上网随便找一个下载命令发现好不靠谱，所以仔细找了一下实用的。

PIP安装

  • 1.先检测pip module是否存在 如果输出数据就表示存在该模块，否则就通过python -m ensurepip安装，之后pip install --upgrade pip更新一下（不搞也行，只是以防万一）
  • 2.pip install xxx即可下载。如果报错，大概是没有把pip路径添加到环境变量里面，所以更换python -m pip install xx。
  • 3.卸载就是pip uninstall xx
  • 4.timeout问题：增加参数由官网下载变为国内镜像下载：pip install xx -i URL --trusted-host xxx.com（xxx.com就是URL里面的网址）
