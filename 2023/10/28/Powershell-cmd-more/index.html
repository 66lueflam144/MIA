PowerShell

PowerShell 是一种跨平台的任务自动化解决方案，由命令行 shell、脚本语言和配置管理框架组成，所有输入和输出都是.NET对象。 PowerShell 在 Windows、Linux 和 macOS 上运行。

功能：

  • 通过函数、类、脚本、模块进行扩展
  • 便于输出的可扩展格式系统
  • 用于创建动态类的可扩展类型系统
  • 对常用数据格式的内置支持

Shell

shell的功能：

  • 命令行记录
  • tab自动补全、命令预测
  • 支持命令和参数别名
  • 链接命令的管道
  • 控制台帮助系统

区别

owerShell 是新式命令 shell，其中包括其他常用 shell 的最佳功能。 与大多数仅接受并返回文本的 shell 不同，PowerShell 接受并返回 .NET对象。

PowerShell Versions

Windows PowerShell ISE //集成脚本环境
