Network Protocols

2023/11/6 LEARN

What is Network Protocol

a set of established rules that specify how to format, send and receive data so that computer network endpoints（including computers, servers, routers and virtual machines）can communicate despite differences in their underlying infrastructures, designs or standards.

In networking, support for protocols can be built into the software, hardware or both.
Without network protocols, computers and other devices would not know how to engage with each other.

TCP/IP model

A set of cooperating network protocols is called a protocol suite.

Transmission Control Protocol/Internet Protocol suite,主要用于客户端-服务端模型，包含了很多协议，几乎覆盖所有层。

Unlike the OSI model, the TCP/IP suite consists of four layers, each with its protocols.

  • Application layer：SMTP、FTP
  • Transport layer
  • Internet layer: IP、ARP、ICMP
  • Network access layer

  • TCP：使用一组规则在信息包级别与其他节点进行交换

  • User Datagram Protocol: 建立应用和网络间低延迟和损失容忍的连接。一个TCP替代通信协议

  • IP：用一组规则发送和接收IP level的消息
