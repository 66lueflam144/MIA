1.sorted，进行排序

def min(arr):
    return sorted(arr)[0]

def max(arr):
    return sorted(arr)[-1]

这段代码非常简洁且有效。它使用了 Python 的 sorted 函数和列表索引来找到数组中的最小值和最大值。

  • sorted(arr)[0] 这段代码首先将数组排序，然后返回排序后的第一个元素，也就是最小值。

  • sorted(arr)[-1] 这段代码同样首先将数组排序，然后返回排序后的最后一个元素，也就是最大值。

需要注意的是，虽然这段代码非常简洁，但它的时间复杂度为 O(n log n)，其中 n 是数组的长度。这是因为 sorted 函数的时间复杂度为 O(n log n)。如果数组非常大，那么这段代码可能会比使用线性扫描（时间复杂度为 O(n)）的方法慢。但是，对于小到中等大小的数组，这段代码的运行速度应该是可以接受的。

作为比较还有一个：

def minimum(arr):
    return min(arr)

def maximum(arr):
    return max(arr)

它使用了 Python 的内置函数 min 和 max 来找到数组中的最小值和最大值。

min(arr) 这段代码返回数组中的最小值。

max(arr) 这段代码返回数组中的最大值。

这两个函数的时间复杂度都是 O(n)，其中 n 是数组的长度，因为它们都需要遍历整个数组一次。这比使用 sorted 函数的方法（时间复杂度为 O(n log n)）要快。所以，对于任何大小的数组，这段代码都能以最快的速度找到最小值和最大值。

join()

def count_sheep(n):
    return ''.join([str(i) + ' sheep...' for i in range(1, n + 1)])

这个函数使用了 Python 的列表推导式（list comprehension）和 join 函数。[str(i) + ' sheep...' for i in range(1, n + 1)] 这段代码将会为从1到n的每个整数生成一个字符串，字符串的内容是这个整数后面跟着 “ sheep…”。然后，join 函数将这些字符串连接成一个大的字符串。

如果我们调用 count_sheep(3)，那么返回的结果将会是 "1 sheep...2 sheep...3 sheep..."。

一个对比

def count_sheep(n):
    sheep=""
    for i in range(1, n+1):
        sheep+=str(i) + " sheep..."
    return sheep

def calculate_century(year):
    if year % 100 == 0:
        century = year // 100
    else:
        century = year // 100 + 1
    return century

# 测试
print(calculate_century(2000))  # 输出：20
print(calculate_century(2001))  # 输出：21

在Python中，//操作符用于执行整数（向下取整）除法。当你使用//操作符，结果将是商的整数部分，忽略了余数。例如，5 // 2将返回2，而不是2.5。

Code 1:

def count_positives_sum_negatives(arr):
    return [len([x for x in arr if x > 0])] + [sum(y for y in arr if y < 0)] if arr else []

This function takes a list arr as input and returns a new list with two elements: the count of positive numbers and the sum of negative numbers in arr. If arr is empty, it returns an empty list. Here’s how it works:

  • [x for x in arr if x > 0] is a list comprehension that creates a new list with all the positive numbers in arr.
  • len([x for x in arr if x > 0]) counts the number of positive numbers in arr.
  • [sum(y for y in arr if y < 0)] calculates the sum of all the negative numbers in arr.
  • if arr else [] checks if arr is not empty. If arr is empty, it returns an empty list [].

Code 2:

def count_positives_sum_negatives(arr):
    return [sum(n > 0 for n in arr), sum(n for n in arr if n < 0)] if arr else []

This function does the same thing as the first one, but it uses a different approach to count the positive numbers:

  • sum(n > 0 for n in arr) uses a generator expression (n > 0 for n in arr) to create a sequence of boolean values (True for positive numbers and False for non-positive numbers). Since True is equivalent to 1 and False is equivalent to 0 in Python, sum(n > 0 for n in arr) effectively counts the number of positive numbers in arr.
